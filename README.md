# Finance Backend

A Node.js backend application for finance-related services.

## Project Structure

```
src/
├── config/
├── controllers/
├── middleware/
├── routes/
├── models/
├── utils/
├── app.js
└── server.js
```

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

For development:
```bash
npm run dev
```

## Usage

The server will start on port 3000 by default.
